Stock Price Prediction 
Stock Price Prediction Web Application with Chatbot
Overview
This repository contains a Stock Price Prediction Web Application built using Python and Flask.
The application analyzes historical stock market data and generates future price predictions using machine learning models.

In addition, the system includes an queries base chatbot that allows users to ask questions related to stock data, predictions, and application usage through a conversational interface.

This project is developed for learning, experimentation, and portfolio demonstration.

 Features
* User authentication (login and registration)
* Fetches real-time and historical stock market data
* Predicts future stock prices based on historical trends
* Interactive charts and visual analytics
* Saves prediction results to the database
* Export predictions in CSV or PDF format
* AI-powered chatbot to answer user queries
* Clean and responsive web interface

  Tech Stack
* Backend: Python, Flask
* Machine Learning: Time-series prediction models
* Chatbot:Query-based intelligent response system
* Database: Flask-SQLAlchemy
* Frontend: HTML, CSS, JavaScript
* Visualization: Interactive charts and graphs
